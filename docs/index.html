Installing Ports (Click to expand)

Clicking a download button will download a zip archive of the port, which you can then copy to the PortMaster autoinstall folder for your platform.

AmberELEC, ROCKNIX, uOS, Jelos: /roms/ports/PortMaster/autoinstall/
muOS: /mmc/MUOS/PortMaster/autoinstall/
ArkOS: /roms/tools/PortMaster/autoinstall/
Knulli: /userdata/system/.local/share/PortMaster/autoinstall/

Once copied, running the PortMaster application will install the port for you. For ports requiring external data, please see the port's README.md by clicking on the blue Details button for the port in question.

Tip: For a more modern, user-friendly experience, consider using Pharos. Pharos automatically handles repository parsing, downloads, and installation.
