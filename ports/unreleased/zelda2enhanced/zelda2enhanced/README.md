## Installation
Download the .zip from https://hoverbat.itch.io/ziiaol and put in the `ports/ziiaol/assets` folder.

## Controls
| Button | Action |
|--|--|
|START|Menu|
|SELECT|Cast Spell|
|D-PAD|Move|
|A|Jump|
|X|Attack|
|L2|Options Menu|

## Notes
This 64-bit version unreleased due to a palette issue. Presumably GMS 1.4.9999 armhf decodes colors using BGR, and aarch64 decodes using RGB. No solution found yet.
