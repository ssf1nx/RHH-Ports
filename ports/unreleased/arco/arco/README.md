## Installation
Purchase the game on Steam and copy `arco.exe` to `ports/arco`.

## Notes
This unreleased port recompresses audio for the game, reducing the memory footprint. It demands around 600MB of RAM at the title screen, so given cfw overhead 1GB devices might be right on the cusp. Test at your own risk.
