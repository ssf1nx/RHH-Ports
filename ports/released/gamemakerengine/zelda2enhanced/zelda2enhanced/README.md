## Installation
Download the 1.0.0.2 zip from https://hoverbat.itch.io/ziiaol and extract it in the zelda2enhanced/assets folder.

## Controls
| Button | Action |
|--|--|
|START|Menu|
|SELECT|Cast Spell|
|D-PAD|Move|
|A|Jump|
|X|Attack|
|L1|Options Menu|
