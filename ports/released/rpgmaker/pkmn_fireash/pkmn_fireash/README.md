# Installation
Download the [Windows bundle](https://pokemonfireash.net/windows/) and add the zip file to the `pkmn_fireash` folder. On first launch the game will be unzipped for you.

# Modifications
The following edits to `mkxp.json` may enhance the experience:

```
"fullscreen": true,
"smoothScaling": true,
"subImageFix": true,
"enableBlitting": false,
"midiSoundFont": "soundfont.sf2",
```
