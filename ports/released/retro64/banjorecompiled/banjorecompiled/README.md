## Installation
Place your `Banjo-Kazooie v1.0` rom with md5 `b29599651a13f681c9923d69354bf4a3` in the `banjorecompiled` folder.

## Requirements
As of v1.0 release this port requires Vulkan 1.2 minimum. The launchscript will check this for you, but you can also check if you have vulkan with the following command and if you have it, check its version: `vulkaninfo 2>/dev/null | grep -m 1 "apiVersion"`.

## Mods
Several mods are being released for this recomp. You may find a few below:

- [Jiggies of Time](https://github.com/Mr-Wiseguy/JiggiesOfTimeRecomp/releases/latest)
- [ BK Nostalgia 64](https://github.com/Mr-Wiseguy/Nostalgia64Recomp/releases/latest)
- [BK Reloaded Texture Pack](https://evilgames.eu/texture-packs/bk-reloaded.htm#rt64)

A note: full game conversion mods such as Kurko's are not yet functional on arm64.
