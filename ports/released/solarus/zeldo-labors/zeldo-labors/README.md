## Installation
Download the game from https://www.solarus-games.org/games/the-labors-of-zeldo/ and put the .solarus file in the `ports/zeldo-labors` folder.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Interact|
|B|Attack|
|X|Item 1|
|Y|Item 2|
|START|Menus|
